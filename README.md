# Password Strength Analyzer (C++)

## Description
Password Strength Analyzer is a simple cyber security tool written in C++ that checks the strength of a password and provides suggestions to improve it.

## Features
- Checks password length
- Detects uppercase letters
- Detects lowercase letters
- Detects digits
- Detects special characters
- Shows password strength (Weak / Medium / Strong)
- Gives security suggestions

## Technologies Used
- C++
- Standard Library

## How to Run

1. Compile the program

g++ password_checker.cpp -o password_checker

2. Run the program

./password_checker
